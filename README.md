# AI 买车导购
Use Java to develop a easy ai application. Help me know about spring ai. 一个spring ai的练手项目。

### 启动说明

- 需要自行申请qwen3和百度车型识别的key，然后修改后端`application.yml`里的key

- mvn编译src下的后端，并运行

- npm编译frontend下的前端，并运行

- 内网穿透需要修改`vite.config.js`里的allowedHosts
